# Course Manager System (C++ Project)

## 📌 Project Overview
The **Course Manager System** is a console-based application developed in **C++** as a first semester project. It is designed to manage students, teachers, and courses efficiently using basic programming concepts.

This system allows administrators to manage all data while users can only view and enroll in courses based on their access level.

---

## ⚙️ Features

### 🔐 Login System
- Secure sign-in using **ID and Password**
- Separate access for **Admin** and **User**

---

### 👨‍💼 Admin Panel
- Add new students and teachers
- Update student/teacher details
- Delete records
- Enroll students in courses
- Manage course data
- Full system control

---

### 👨‍🎓 User Panel
- View available courses
- Enroll in courses
- View enrolled courses

---

## 🧠 Concepts Used
This project is built using the following C++ concepts:

- Variables and Data Types
- Loops (for, while, do-while)
- Conditional Statements (if, if-else)
- Functions
- Structures
- File Handling (read/write/update data)
- Console screen clearing (system("cls"))

---

## 💾 File Handling
All data (students, teachers, courses, login info) is stored using file handling to ensure data persistence even after program exit.

---

## 🚀 Purpose of Project
This project was created as a beginner-level system to understand how real-world management systems work using basic programming logic in C++.
